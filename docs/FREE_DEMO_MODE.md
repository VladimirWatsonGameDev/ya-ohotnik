# Бесплатный демо-режим

## Зачем

Демо-режим нужен, чтобы быстро проверить игру в VK Mini Apps без расходов на backend, платежи и серверную инфраструктуру.

## Что включено

- Игровые уровни.
- Сохранение прогресса через VK Storage или `localStorage` вне VK.
- Магазин винтовок за внутриигровые монеты.
- Кнопка `Получить 1000 монет` в магазине.
- Покупка винтовок за монеты.

## Что отключено

- Реальные платежи.
- Покупка монет за деньги.
- Серверная проверка баланса.
- Серверный рейтинг.

## Как вернуть платежи позже

1. Заменить демо-кнопку в `src/game/scenes/ShopScene.ts` на вызов `vk.buyCoins(packId)`.
2. Указать реальный `VITE_VK_APP_ID` в `.env`.
3. Настроить товары в кабинете VK.
4. Добавить backend для проверки callback от VK.
5. Начислять монеты только после подтвержденной оплаты.
