# Я Охотник

Аркадная VK Mini App игра в духе классической охоты: 100 уровней, дневная и ночная охота, бонусные раунды, рейтинг, магазин оружия и монетизация через покупку монет.

Текущий режим: бесплатный MVP без реальных платежей. В магазине есть демо-кнопка для получения монет.

## Что уже подготовлено

- Техническая структура проекта для VK Mini Apps.
- Алгоритм запуска, предзагрузки и догрузки файлов.
- Конфигурация 100 уровней через генератор баланса.
- Манифест ассетов для дневных, ночных и бонусных сцен.
- Базовые Phaser-сцены: boot, preload, menu, level select, hunt, results, shop.
- Интеграционный слой VK Bridge: init, пользователь, storage, покупки.
- Документы для загрузки, релиза, модерации и дальнейшей разработки.

## Стек

- Vite
- TypeScript
- Phaser
- `@vkontakte/vk-bridge`

## Быстрый старт

```bash
npm install
npm run dev
```

Для релизной сборки:

```bash
npm run build
```

Перед проверкой платежей создайте `.env` по примеру `.env.example` и укажите реальный `VITE_VK_APP_ID`.

Готовые файлы появятся в папке `dist`. Именно ее содержимое загружается на хостинг или в выбранный релизный контур VK Mini Apps.

## Важное перед публикацией

1. Заменить временные SVG-ассеты в `public/assets` на финальные изображения, звуки и анимации.
2. Подключить серверную часть для платежей и проверки покупок.
3. Настроить приложение в кабинете VK Mini Apps.
4. Протестировать запуск в мобильном VK-клиенте, десктопе и webview.
5. Проверить юридические тексты: политика конфиденциальности, пользовательское соглашение, правила покупок.
